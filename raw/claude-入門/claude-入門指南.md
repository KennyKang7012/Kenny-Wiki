# Claude 入門指南

> Source: raw/Claud 入門指南.pdf
> Collected: 2026-04-17
> Published: Unknown

Claude 不是只會回答問題的聊天機器人，它是真正幫你做事的 AI 工作夥伴。只要你設定好目標，它就會想辦法幫你達成。

## 核心特色

- 回覆更自然像真人對話，不確定時不會亂回答
- 深度思考能力，可理解超長文件和大型程式專案
- 可直接幫你把事情做完，不只是告訴你怎麼做

## 適用情境

- 上班族：早晨串聯行事曆列出待辦，會議錄影直接生成會議記錄並同步 Notion
- 學生：150 頁英文論文直接生成論點、研究方法，整理成比較表格並快速製作簡報
- 創作者：把品牌風格存進專案，提供素材後寫出風格一致的文案
- 工程師：列出需求直接幫你開發 + Debug，優化 code、做驗證測試、管理 Git

## 方案比較

| 方案 | 價格 | 特點 |
|------|------|------|
| Free | $0/月 | 使用有限額，預設 Sonnet 4.6，無 Claude Code / Cowork |
| Pro | $20/月 | 5 倍使用量，解鎖 Opus 模型，享 Claude Code / Cowork |
| Max | $100/月起 | 5-20 倍 Pro 使用量，尖峰時段新模型優先使用權 |

推薦：日常管理和程式開發用 Pro；複雜專案開發考慮 Max。

## 模型選擇

- **Haiku**：極速、輕量、省 Token，適合快速問答、簡單分類、基礎資料處理
- **Sonnet**：兼具性能和實用性，CP 值最高，適合寫作、分析、程式開發和日常工作
- **Opus**：處理複雜思考，速度較慢、Token 消耗多，適合超大型專案和長篇文獻

## Prompt 基本結構

- **Role/Context**：給定角色、背景、檔案
- **Task/Target**：清楚說明具體目標
- **Rule/Format**：規範格式、長度、語氣

原則：不要教 AI 怎麼做，而是告訴它「做什麼 & 格式」。目標導向，模型決定步驟和工具。

## Claude 生態系

- **Claude Cowork**：讓 AI 自動操作電腦，在背景執行、自主存取檔案、排程任務
- **Claude Code**：直接運行於 Terminal 的進階 AI 程式助手，全端開發一手包辦
- **MCP**：給 Claude 操作外部工具的手（瀏覽器、Notion、Drive 等）

## Claude Code 進階功能

- **CLAUDE.md**：放在專案根目錄的指令集，每次啟動自動讀取
- **Skills**：封裝好的 SOP，讓 Agent 成為某領域專家，不佔用 Context
- **Subagents**：平行處理複雜任務的臨時工，每個有獨立上下文
- **Hooks**：系統層級自動化，條件達成後自動觸發動作

## 常用指令

| 指令 | 功能 |
|------|------|
| /compact | 壓縮上下文，避免 Context 爆滿 |
| /account | 查看使用額度 |
| /init | 初始化專案記憶，建立 CLAUDE.md |
| /clear | 清除當前對話 |

| /model | 切換模型（Sonnet / Haiku / Opus） |
