# Claude 模型方案比較

> Sources: Wilson AI Lab, Unknown
> Raw: [claude-入門指南](../../raw/claude-入門/claude-入門指南.md)

## 定義
Claude 提供三種付費方案（Free / Pro / Max）和三種模型（Haiku / Sonnet / Opus），選對組合才能在成本和效能間取得最佳平衡。

## 重點整理

**方案比較**

| 方案 | 價格 | 適合對象 |
|------|------|----------|
| Free | $0/月 | 輕量體驗，有使用限額，無 Claude Code / Cowork |
| Pro | $20/月 | 最推薦，5 倍使用量，解鎖 Opus 模型，享 Claude Code |
| Max | $100/月起 | 重度使用者，5-20 倍 Pro 使用量，尖峰時段優先 |

- 日常管理和程式開發 → 推薦 Pro
- 複雜專案開發 → 考慮 Max
- 免費版無法使用 Claude Code 和 Cowork

**模型選擇**

| 模型 | 特性 | 適用場景 |
|------|------|----------|
| Haiku | 極速、輕量、省 Token | 快速問答、簡單分類、基礎資料處理 |
| Sonnet | 兼具性能和實用性，CP 值最高 | 寫作、分析、程式開發和日常工作（主力首選） |
| Opus | 深度思考，速度較慢，Token 消耗多 | 超大型專案、程式架構、長篇文獻 |

**使用對象參考**
- 上班族：串聯行事曆、生成會議記錄、同步 Notion
- 學生：論文論點生成、比較表格製作、簡報製作
- 創作者：品牌風格文案（存進專案後自動套用）
- 工程師：開發 + Debug + 驗證測試 + Git 管理
