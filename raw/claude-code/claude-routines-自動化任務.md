# Hei.AI — Claude Routines 新功能：讓 AI 每天自動幫你做事

## Routines 是什麼

Routines 是讓 Claude 定時自動執行任務的功能，像是「會思考的鬧鐘」。

**響完後 Claude 會自動：**
- 上網搜資料
- 分析內容、寫摘要
- 推送到 Notion、Slack、Gmail
- 處理完整個工作流程

**三種觸發方式：**
1. 定時排程 — 每小時、每天、每週固定時間跑
2. API 觸發 — 從外部系統發 HTTP 請求觸發
3. GitHub Event — 當有新 PR 或 Release 時自動跑

整個過程在 Claude 的雲端執行，電腦關機也沒關係（前提是用 Remote 模式）。

## 為什麼值得用

1. **零程式門檻**：不用學 n8n 或 Make，只要用自然語言說需求
2. **時間被放大**：AI 每天幫你做重複工作，省下時間做有判斷力的事
3. **新手最佳入口**：門檻最低的自動化工具

## 7 步搭好一個 Routine

1. **打開 Claude Code**（Desktop App 或 claude.ai）
2. **用自然語言描述需求**，範例：
   - 「每天早上 8 點搜集 AI 資訊，整理好放入 Notion」
   - 「每天檢查 YouTube 頻道有沒有新影片，抓字幕翻譯成繁體中文做摘要」
   - 「每天早上 9 點檢查 Gmail，生成未回覆郵件的回覆草稿」
3. **AI 會問幾個問題**：結果存到哪裡、監控哪些來源、有裝什麼工具
4. **明確要求 Remote 版本**（不說則預設 Local，電腦關機就停）
   - Local：在自己電腦跑，電腦必須開機
   - Remote：在 Claude 雲端跑，電腦關機也執行
5. **AI 自動搭建整個系統**：裝工具、寫腳本、建 Notion DB、設觸發時間
6. **按 Run Now 測試**（必要步驟，第一次執行需授權各項權限）
7. **繼續用自然語言修改**：「幫我加一個股市分析功能」

## 限制

- **場景不夠深入**：複雜多步驟工作流做不好，只有一個 Prompt 控制全部
- **不像 n8n 每步分開控制**：出問題不知道哪步錯了
- **每日執行次數有上限**：Pro 5 次、Max 15 次、Team/Enterprise 25 次
- **仍在 Research Preview 階段**：功能和限制可能會改變

## 觀念轉變

「效率高」不是「自己做得快」，而是「讓 AI 做重複的事，你做判斷和創造」。

你的時間應該花在策略決策、創意發想、人際溝通、學習新東西；重複固定有規則的工作交給 AI 的 Routine 處理。
